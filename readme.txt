Use this plugin to enter and display testimonials.

== Description ==

Showing testimonials on your website helps you to attract and keep new clients/customers! Use this plugin to enter and display testimonials using included widget or shortcodes.

= Features =

* Allows for unlimited testimonials
* Shortcode to show all testimonials
* Shortcode to show a random testimonial
* Included widget shows a random testimonial

**Or**
